# N予備コメビュ

N予備校の生放送授業において、コメント一覧を表示するChrome拡張です。

サーバーなどへ余計なアクセスはせず、描画エリアに表示されたコメントだけが一覧に表示される仕様です。

運営コメントも取得できるようになりました。(v1.1)
動作しなくなったためコメントNo.を取り除いてPIXI.Textをフックする形式に修正。(v1.3)
